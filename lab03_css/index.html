Welcome to CSS Fundamentals

This lab covers essential CSS concepts through practical exercises. Use the navigation above to explore different topics.

Topics Covered

Selectors

Learn about element, class, ID, attribute, pseudo-class, and pseudo-element selectors.

Typography

Master font families, sizes, weights, text alignment, decoration, and shadows.

Backgrounds

Explore background colors, images, gradients, positioning, and blend modes.

Images

Apply borders, shadows, filters, hover effects, and overlays to images.

Buttons

Style buttons with colors, sizes, shapes, states, and transitions.

Forms

Create accessible form styles with focus states and validation visuals.

Cards

Build consistent card components with spacing, borders, and hover effects.

Navigation

Build horizontal and vertical menus with dropdowns and responsive behavior.

Transforms

Use CSS transforms for rotation, scaling, translation, and 3D effects.

Advanced Effects

Explore gradients, animations, loaders, clipped text, and complex shadows.

Portfolio

Practice building a polished portfolio page with layout and styling.

Schedule

Style tables with zebra striping, hover effects, and responsive layouts.

GitHub Pages

Design a clean, modern landing page suitable for GitHub Pages hosting.
